🗺️ 選挙・政治活動マップツール

ブラウザだけで動く、ポスター貼付けとポスティングの進捗管理マップです。サーバ登録不要・データは各自の端末に保存されます。

🗺️ ポスターマップ

選挙運動用ポスターの掲示場(掲示板)を地図に表示し、貼付けの進捗(未・予約・完了・要確認・異常)を色分け管理。選管公表の掲示場一覧CSVを読み込んで使います。

📮 ポスティングマップ

政治活動用ビラのポスティング区域・ルートを地図に描いて、予定・配布中・済を色分け管理。GeoJSONの書出/読込でメンバー間の共有ができます。
⚠️ 使い方の注意: データはブラウザ(端末)ごとの保存です。端末をまたいで共有するときは「書出」したファイルを渡して相手が「読込」してください。ポスターの掲示場データはお住まいの選挙管理委員会が公表する一覧をCSV化して読み込んでください(同梱データはダミーです)。
チームみらい(安野たかひろ陣営)公開の poster-map の仕組みを参考にした独自実装です(コードは書き下ろし)。 ソースコード: GitHub / 地図: 国土地理院・OpenStreetMap
